# Rupeshkumar Bhosale Portfolio

Welcome to my personal portfolio! This is a modern, interactive, and responsive portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**, showcasing my skills, projects, certifications, and experience.

---

## 🌐 Live Demo

[View Live Portfolio](https://rupesh-portfolio-ultp.onrender.com/)  

---

## 💻 Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express, Firebase (for contact form)
- **Database:** MongoDB, Firestore
- **Languages:** JavaScript, TypeScript, Dart, Python
- **Tools:** Git, GitHub, Figma, VS Code

---
## 📂 Sections Overview

### 1. Home
- Animated introduction with gradient text.
- Quick CTA linking to Projects and Contact.

### 2. About
- Personal bio and timeline of education & internships.
- Animated skill badges and fun facts.

### 3. Skills
- Categorized skills:
  - **Frontend, Backend, Mobile, Database, Tools**
- Progress indicators and badges for visual representation.

### 4. Projects
- Interactive project cards with hover effects.
- Detailed modal view for each project:
  - Overview, Tech Stack, Features, Architecture, Challenges.
- "View More Projects" button linking to external project repository or LinkedIn.

### 5. Certifications
- Grid of certifications with clickable images for preview.
- "View More Certificates" button linking to LinkedIn.

### 6. Experience
- Timeline layout with roles, durations, and key achievements.
- Company logos with project highlights.

### 7. Contact
- Contact form with Name, Email, Message.
- Quick links to GitHub, LinkedIn, Email.
- Downloadable Resume option.
